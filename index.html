Introduction

The goal the this project is to create a dedicated chess computer based on tiny ARM computers, the Stockfish chess engine, the DGT eboard and DGT XL Clock, and the dgtnix driver.

Currently the Rikomagic MK 802 II is supported , and I also plan to support the Raspberry Pi

Status

PicoChess v0.1 in working on the 802MK II ARM device. You can see a video of PicoChess running on a standard PC, and running on a MK802 II ARM key :-)

It is not yet released for Raspberry Pi, because there is a nasty bug in the Raspberry Pi USB firmware causing the device to hang.

Done so far :

  • Added DGT XL clock message display to dgtnix driver
  • Added DGT board/clock support to Stockfish
  • Player can choose skill level
  • Player can choose computer thinking time
  • Allow to takeback moves
  • Software shutdown
  • Add opening books, and allow opening book selection

Download and installation

You will have to download install the latest PicoChess image file and write it to a micro SD card :

  • PicoChess MK802II v0.3 (4GB micro SD Card)
  • PicoChess MK802II v0.2 (4GB micro SD Card)
  • PicoChess MK802II v0.1 (8GB micro SD Card)

You will need to unzip the image with 7zip and write it to a suitable SD card using the UNIX tool dd. Windows users should use Win32DiskImager. Do not try to drag and drop or otherwise copy over the image without using dd or Win32DiskImager – it won’t work.

Copying the image to an SD Card on Windows

  1. Insert the SD card into your SD card reader and check what drive letter it was assigned. You can easily see the drive letter (for example G:) by looking in the left column of Windows Explorer. If the card is not new, you should format it; otherwise Win32DiskImager may hang.
  2. Download the Win32DiskImager utility. The download links are on the right hand side of the page, you want the binary zip.
  3. Extract the executable from the zip file and run the Win32DiskImager utility. You may need to run the utility as Administrator.
  4. Select the picochess-MK802II-vx.x.img image file you extracted earlier
  5. Select the drive letter of the SD card in the device box. Be careful to select the correct drive; if you get the wrong one you can destroy your computer's hard disk!
  6. Click Write and wait for the write to complete.
  7. Exit the imager and eject the SD card.
  8. Insert the card in the MK802II, connect it to a DGT eboard and XL clock, power it on, and it should boot up with a nice 'Hello' message on the clock after 1-2 minutes.
  9. Play !
