PicoChess

Stand alone chess computer based on Stcokfish and DGT board

Download Download View PicoChess on GitHub

Introduction

The goal the this project is to create a dedicated chess computer based on tiny ARM computers, the Stockfish chess engine (http://www.stockfishchess.com/), the DGT eboard and DGT XL Clock (http://www.dgtprojects.com/), and the dgtnix driver (http://dgtnix.sourceforge.net/).

Currently the Rikomagic MK 802 II is supported (http://store.cloudsto.com/rikomagic/rikomagic-mk802-ii-detail.html), and I also plan to support the Raspberry Pi (http://www.raspberrypi.org/)

Status

You can see a video here running on a standard PC : http://www.dailymotion.com/video/xt2eul_pichess_tech

And running on a MK802 II ARM key :-) : http://www.dailymotion.com/video/xt4b85_pichess2_tech

The inital proof of concept is working on a standard Linux PC and an 802MK II ARM device. There is a nasty bug in the Raspberry Pi USB firmware causing the device to hang (http://www.raspberrypi.org/phpBB3/viewtopic.php?f=44&t=8010&p=158380). I hope this will be solved in few weeks :-)

Done so far :

  • Added DGT XL clock message display to dgtnix driver
  • Added DGT board/clock support to Stockfish
  • Player can choose skill level
  • Player can choose computer thinking time
  • Allow to takeback moves
  • Software shutdown
